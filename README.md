Se requieren las librerias de Python: "pygame", "math", "random", "argparse", "sys" y "time".
Para ejecutar, usar Python 2.x y seguir estos pasos:

```console
source TP/Bin/activate
python draw.py
```

Para salir del *virtualenv*:

```console
deactivate
```
